# ecommerce-website-php
A simple e-commerce website built using HTML, CSS, JavaScript, and PHP as a web design project.
## E-Commerce Website (PHP)

This is a multi-page e-commerce website built as a web design final project.

### Technologies Used
- HTML
- CSS
- JavaScript
- PHP

### Features
- Reusable header, navigation, and footer using PHP includes
- Product pages with interactive attributes
- Dropdown navigation menu
- Contact form with character limit validation
- Site-wide dismissible notice banner

This project was developed for learning and demonstration purposes.
